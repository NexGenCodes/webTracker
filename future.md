# Future Features & Roadmap

## WhatsApp Bot
- [ ] **Document Manifest Support**: Handle .csv, .txt, and .pdf uploads for bulk shipment creation.
- [ ] **Advanced AI Context**: Improve parsing of edge-case addresses and specific regional phone formats.
- [ ] **Interactive Commands**: Support for list-based menus for shipment selection.

## Backend Infrastructure
- [ ] **CI/CD Optimization**: Faster Docker builds and automated deployments to VPS.
- [ ] **Telemetry Dashboard**: Real-time stats on parsing success/failure and bot vitals.
- [ ] **Multi-Carrier Integration**: Automatically fetch tracking updates from external carriers.

## Admin Dashboard
- [ ] **Shipment Analytics**: Visual charts for shipment trends and delivery performance.
- [ ] **Bulk Actions**: Enhanced UI for managing hundreds of shipments simultaneously.
- [ ] **Internationalization**: Support for more languages beyond the current set.
